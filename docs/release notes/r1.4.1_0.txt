Objeck v1.4.1_0

This release provides support for runtime introspection.  New classes where added to support introspection along with the addition of the 'TypeOf' method, which is available to all objects.  In addition, minor bug fixes where made to improve stability.

The Objeck programming language is a light-weight object-oriented programming language with functional features. The programming system consists of a compiler, virtual machine (with garbage collector and JIT compiler) along with a command line debugger.
